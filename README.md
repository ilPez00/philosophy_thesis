## What This Is

AYU is a three-layer computational system designed around a single thesis:

> **Passive observation → causal understanding → intentional action → measurable life improvement.**

It is not a productivity app. It is not a journal. It is an attempt to make the gap between *wanting* and *doing* legible — first to a machine, then to the person living it.

## Philosophical Lineage

The system sits at the intersection of several traditions:

- **Extended Mind** (Clark & Chalmers, 1998) — cognition is not skin-bound. Aura's wiki is memory that lives outside the skull.
- **Philosophy of Action** (Aristotle's *Praxis*) — action undertaken for its own sake, not merely for an outcome. The name is not accidental.
- **Phenomenology** (Husserl's *epoché*) — Aura's AR overlay attempts the radical: bracket the device interface entirely, leaving only the world + the agent's intentions projected onto it.
- **Federated Will** — each person's goals are sovereign. The community validates effort, not outcomes. No central authority judges whether a goal is worth pursuing.
- **Causal, Not Reactive AI** — current AI systems answer. AYU aims for the prior step: building a *causal model* of a person's life before any answer is needed.

## Current Implementation Status

All three layers are fully implemented and running:

- **Aura** — Android app (Kotlin/Jetpack Compose), deployed on device, keyless via free LLM proxies
- **Rachmaninov** — shared ontology (14 domains, 130+ visual hints), live sync endpoint, ACT feedback loop
- **Praxis** — full-stack webapp (React/Express/Supabase), deployed on Railway + Vercel, Axiom AI agent, Lattice physical device network
